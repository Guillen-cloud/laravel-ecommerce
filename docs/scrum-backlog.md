# Scrum Backlog - Laravel Ecommerce

## Product Goal

Construir una tienda ecommerce Laravel inspirada en tiendas retail, con panel admin, carrito, checkout y facturacion PDF.

## Sprint 1 (en curso)

### Historias

1. Como visitante quiero ver una pagina de inicio comercial para explorar productos destacados.
2. Como visitante quiero filtrar el catalogo sin recargar pagina para encontrar productos mas rapido.
3. Como visitante quiero ver detalle y stock por sucursal para decidir mi compra.

### Tareas completadas

- [x] Configurar rutas MVC para home y catalogo.
- [x] Implementar HomeController y ProductController.
- [x] Construir layout Bootstrap base.
- [x] Crear vista de home y catalogo.
- [x] Implementar filtros AJAX en catalogo.
- [x] Crear vista de detalle de producto.
- [x] Integrar base de facturacion PDF con DomPDF.
- [x] Modulo carrito con AJAX.
- [x] Checkout basico y creacion de orden.
- [x] Persistencia carrito por usuario.

### Tareas siguientes

- [x] Panel admin CRUD productos.
- [x] Gestion de stock por sucursal.

- [x] Permisos admin en rutas.

## Definition of Done

- Codigo funcional con rutas navegables.
- Sin errores de analisis estatico.
- Cambios registrados en bitacora.
- Commit con mensaje claro por incremento.
